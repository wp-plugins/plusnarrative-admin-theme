== Description ==

The PlusNarrative Admin Theme plugin easily allows users to access critical information from PlusNarrative.

The plugin does the following:
*   Adds a PlusNarrative website link to the admin bar
*	Cleans up the dashboard to highlight important information
*	Brands the login page
*	Adds additional dashboard widgets
*	Displays the latest PlusNarrative blog posts

== Installation ==

1. Upload `pn-admin-theme.zip` to the `/wp-content/plugins/` directory
2. Unarchive `pn-admin-theme.zip`
3. Activate the plugin through the 'Plugins' menu in WordPress
